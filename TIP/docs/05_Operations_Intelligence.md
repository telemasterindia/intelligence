# Operations Intelligence

Operations Intelligence explains what is happening inside daily work.

## Goals

- Track import status and operational exceptions.
- Monitor campaign readiness.
- Preserve call and lead outcomes over time.
- Identify rework, bad data, delays, and process bottlenecks.
- Help managers choose where attention is needed today.

## Core Signals

- Import batch status.
- Validation severity.
- Lead outcome history.
- Campaign activity.
- Agent outcome patterns.
- Geography and calling-window performance.
- Vendor quality trends.

## Operational Questions

TIP should help answer:

- Which imports are pending, failed, or ready?
- Which campaigns have enough clean data?
- Which vendors are creating validation problems?
- Which outcomes are changing by campaign?
- Which time windows are producing better contact rates?
- Which operational issues repeat every week?

## Boundaries

Operations Intelligence should not calculate final business profitability by itself. It provides operational evidence that Business Intelligence and Recommendation modules can use.
