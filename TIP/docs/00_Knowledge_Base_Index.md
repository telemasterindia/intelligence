# TIP Knowledge Base Index

This legacy index is retained for continuity. The Version 1.0 master index is [README](README.md).

This folder is the stable Product Bible for TIP, the TeleMaster Intelligence Platform.

TIP is an internal, local-first Operations Intelligence Platform for lead ingestion, operational history, business analysis, and future recommendations. It is not a CRM, dialer, billing system, public SaaS product, or generic lead manager.
